# アプリ名: サブスクーラー


アプリ説明:

  * 利用しているサブスクリプションサービスの月額の利用料金を可視化してを管理サポートできるWebアプリケーションです

主な機能:
  * 目玉機能は以下の2つとなります。
    ・サブスクリプションサービスの月額利用料金を可視化する機能
    ・当アプリの利用者のサービス登録情報を集計し、性別や年代などで流行しているサービスの統計を利用できるAPI

使用技術
  * バックエンド
    Elixir
    Phoenix Webフレームワーク 等
  * フロントエンド
    JavaScript
    CSS 等
